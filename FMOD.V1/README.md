# FMOD.V1

This project contains wrapping code and the native libraries for FMOD `1.10.20`.

## Supported Platforms

* `win-x86`
* `win-x64`
* `osx-x64`
* `linux-x86`
* `linux-x64`
* `linux-arm`
* `android-x86`
* `android-x64`
* `android-arm64`
* `android-arm`
